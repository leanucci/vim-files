h1. Usage

In your home directory, run:
<pre><code>
git clone git://github.com/miloops/vim-files.git .vim

</code></pre>

Then:
<pre><code>
ln -s ~/.vim/vimrc .vimrc
ln -s ~/.vim/gvimrc .gvimrc
</code></pre>

This is my adaptation to miloops "vim-files":http://github.com/miloops/vim-files/tree/master
